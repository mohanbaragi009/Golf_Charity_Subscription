# **App Name**: Golf Charity Platform

## Core Features:

- User Subscription & Payments: Enable users to subscribe, select monthly or yearly plans, and manage their payment details securely.
- Golf Score Entry: Provide an intuitive interface for users to enter their latest Stableford golf scores.
- Performance Tracking Dashboard: Display personalized golf performance history, statistics, and trends to users.
- Charity Selection & Allocation: Allow users to choose a specific charity and view how their subscription portion contributes to it.
- AI-Powered Prize Draw Tool: An intelligent algorithm tool that processes user data (participation, scores) to determine fair and engaging monthly prize winners.
- Admin Portal: A comprehensive web interface for administrators to manage users, subscriptions, charities, prize draws, and system settings.

## Style Guidelines:

- The chosen color scheme is light to convey a fresh, modern, and engaging feel. The primary color is a rich, modern green (#1FA343), symbolizing nature and growth, suitable for main branding and interactive elements. The background color is a very light, desaturated green (#EEF6EF) for a clean and serene base. An accent color, a vibrant lime green (#8AF533), will be used to highlight calls to action and draw attention to key information, providing energetic contrast.
- All text will use 'Belleza', a humanist sans-serif font. Its refined and distinctive personality aligns with the app's modern and engaging aesthetic, making it suitable for both headlines and smaller chunks of text like descriptions and labels without overwhelming the user.
- Use minimalist, vector-based icons with a modern geometric style. Icons related to golf (e.g., ball, club, flag), achievements (e.g., trophy, medal), and charity (e.g., heart, hand) should be clear, clean, and non-traditional.
- Implement a clean, card-based layout to organize information such as scores, charities, and prize draw details. Emphasize responsive design to ensure optimal viewing and interaction across various devices, with ample whitespace for clarity and visual comfort.
- Incorporate subtle and smooth animations for transitions between sections, score submission confirmations, and a dynamic visualization during prize draws to enhance user engagement without causing distraction.
